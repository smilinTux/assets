## Usage

### In Projects

Reference assets in your project documentation:

```markdown
![SKStacks Logo](https://raw.githubusercontent.com/smilinTux/assets/main/svg/logos/skstacks-wordmark.svg)

![King Divad](https://raw.githubusercontent.com/smilinTux/assets/main/mascots/king-divad/king-divad.svg)
```

### In CSS

```css
@import url('https://raw.githubusercontent.com/smilinTux/assets/main/css/themes/smilinTux-theme.css');
```

### In HTML

```html
<link rel="stylesheet" href="https://raw.githubusercontent.com/smilinTux/assets/main/css/themes/smilinTux-theme.css">
```

## Brand Guidelines

### Colors

| Swatch | Name | Hex | Usage |
|--------|------|-----|-------|
| | **Primary Deep Blue** | `#0E2B3D` | Serious foundation, headings |
| | **Accent Aqua** | `#00C8D4` | Curiosity & tech, links, highlights |
| | **Penguin Yellow** | `#FFD166` | Happy accents, callouts |
| | **Ice White** | `#F7FBFF` | Backgrounds |
| | **Penguin Black** | `#0B0B0B` | Body text, contrasts |
| | **Soft Gray** | `#A9B2BB` | Muted UI, secondary text |

### Typography

- **Headlines**: Inter or Poppins — geometric, friendly
- **Body**: Source Sans 3 — readable, professional
- **Code**: IBM Plex Mono or Fira Code

### Logo Usage

- Maintain aspect ratio at all times
- Use appropriate sizing for context
- Ensure sufficient contrast (>4.5:1 where text overlays)
- Respect minimum clear space around logos

## Mascot Guidelines

### King Divad
- **Personality**: Wise, friendly, curious — the happy-feet penguin monarch
- **Usage**: Official communications, documentation headers, community presence
- **Variants**: Full illustration, icon-only, monochrome
- **Rules**: Do not alter mascot expressions; crown tilt reserved for official statements

### Pengu Friends
- **Personality**: Playful, helpful, diverse
- **Usage**: Community content, tutorials, examples
- **Variants**: Multiple characters for different contexts
